# C10TeamViewer

This application allows you to input various data points to compile a rough roster of your 'team'.  This includes management, an engineer and an intern.

# User Needs

<img width="724" alt="image" src="https://user-images.githubusercontent.com/105176171/179372363-4053649c-7136-4afd-b32c-4c150cb1e2bb.png">

This application allows the user to generate an HTML team roster that contains basic information via the command line.   

# Installation 

Clone the Github repository and proceed to run node index.js and follow the prompts to generate a quality team viewer page in HTML.
